# AESA Backend

Backend NestJS V1 para el Sistema de Autorizaciones Frio/Calor AESA.

## Instalacion local

```bash
npm install
```

## Variables de entorno

 `.env` y completar los datos locales.

## Arranque

```bash
npm run start:dev
```

## Modulos incluidos

- `auth`
- `users`
- `requests`
- `documents`
- `printing`
- `history`
- `whatsapp`
- `common`
